    什么是zeppelin？

    Apache Zeppelin 是一个让交互式数据分析变得可行的基于网页的开源框架。
    Zeppelin提供了数据分析、数据可视化等功能。
    Zeppelin 是一个提供交互数据分析且基于Web的笔记本。
    方便你做出可数据驱动的、可交互且可协作的精美文档，并且支持多种语言，包括 Scala(使用 Apache Spark)、Python(Apache Spark)、SparkSQL、 Hive、 Markdown、Shell等等。
    – 百度百科-Zeppelin

    启动zeppelin

    用容器启动

    docker run -tid –name zeppelin -v zeppelin-home:/zeppelin/notebook -p 9999:8080 apache/zeppelin:0.8.0

    二进制文件启动
    1. 下载安装包，下载地址
    2. 解压tar xf zeppelin-0.8.0-bin-all.tgz得到”zeppelin-0.8.0-bin-all”, zeppelin运行环境需要jdk 1.7以上版本，jdk安装可以参考Jdk 1.8安装
    3. 修改”zeppelin-0.8.0-bin-all/conf/zeppelin-site.xml.template”，将template后缀去掉cp zeppelin-site.xml.template zeppelin-site.xml，可以修改里面的配置项

    4. 运行

    5. 访问Ip:8080
